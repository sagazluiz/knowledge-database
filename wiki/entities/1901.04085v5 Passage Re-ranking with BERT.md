# Passage Re-ranking with BERT

## Resumo Executivo

Este documento descreve uma abordagem para reclassificação de passagens usando o modelo BERT. A técnica envolve a utilização do BERT como um modelo de linguagem para avaliar a relevância das passagens em relação a uma consulta.

## Componentes e Arquitetura Técnica

A arquitetura técnica deste método consiste nos seguintes componentes:

1. **BERT (Bidirectional Encoder Representations from Transformers):** Um modelo de transformador que é treinado para entender o contexto de uma palavra em relação às palavras ao seu redor.
2. **Passage Retrieval:** Uma etapa que envolve a recuperação das passagens relevantes a partir de um índice inverso pré-construído.
3. **Re-ranking:** Uma etapa que utiliza o modelo BERT para reclassificar as passagens recuperadas com base na sua relevância em relação à consulta.
