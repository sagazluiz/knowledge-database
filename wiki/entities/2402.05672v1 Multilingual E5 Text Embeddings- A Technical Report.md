# ARQUITETANDO SISTEMAS AGENTES

## Resumo Executivo

Este documento descreve a arquitetura técnica do projeto "Arquitetando Sistemas Agentes". A tecnologia utiliza agentes para monitorar e gerenciar sistemas de forma autônoma.

## Componentes e Arquitetura Técnica

A arquitetura técnica do projeto consiste nos seguintes componentes:

* **Agentes:** Os agentes são os principais atores da arquitetura. Eles são responsáveis por monitorar e gerenciar sistemas de forma autônoma.
* **Plataforma de Agentes:** A plataforma de agentes é onde os agentes residem e interagem entre si e com outros sistemas.
* **Sistema Gerenciador:** O sistema gerenciador é responsável pelo controle centralizado dos agentes e pela coordenação das atividades deles.
